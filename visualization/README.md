# ngk visualization

A small React Three Fiber playground for the `ngk` geometric kernel. It's a
collection of tiny "experiment" components that each exercise one piece of the
kernel — a sidebar switches between them.

## Prerequisites

- Rust toolchain with the `wasm32-unknown-unknown` target:
  `rustup target add wasm32-unknown-unknown`
- [`wasm-pack`](https://rustwasm.github.io/wasm-pack/): `cargo install wasm-pack`
- Node.js 18+

## Run

From `visualization/`:

```bash
npm install
npm run dev
```

`npm run dev` first rebuilds the wasm bundle into `visualization/src/wasm/` and
then starts Vite.

## Adding a new experiment

1. Copy `src/experiments/_template/` to `src/experiments/<YourName>/`.
2. Register it in `src/experiments/registry.ts`:

   ```ts
   {
     id: "your-name",
     title: "Your experiment",
     group: "NURBS", // or another group
     component: lazy(() => import("./YourName")),
   }
   ```

3. The component is rendered as a child of the shared R3F `<Canvas>`
   (see `components/SceneShell.tsx`), so return three.js JSX.

## Project layout

```
src/
  App.tsx                  # shell: sidebar + viewport + URL-hash routing
  components/
    SceneShell.tsx         # Canvas + lighting + grid + orbit controls
    ExperimentSidebar.tsx
  kernel/
    useKernel.ts           # async init of the wasm module, cached
    nurbs.ts               # tiny TS helpers around the wasm API
  experiments/
    registry.ts            # list of experiments
    NurbsCurveEditor/
    _template/
  wasm/                    # generated by wasm-pack (gitignored)
```
